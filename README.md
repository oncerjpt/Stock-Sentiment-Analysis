# Stock-Sentiment-Analysis-Using-Machine-Learning
`1. Developed a machine learning pipeline to analyse financial news sentiment and processed stock-related news headlines for Tesla and META from FINVIZ by parsing downloaded HTML datasets.`
`2. Implemented sentiment analysis using NLTK VADER and trained multiple ML models and Neural Networks which includes Logistic Regression, SVM, Random Forest, Gradient Boosting, KNN, and Decision Trees.`
